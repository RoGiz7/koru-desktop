# Fuente display — Rajdhani (SIL Open Font License)

La app busca aquí estos 3 ficheros (si faltan, cae a Inter sin romper nada):

- `Rajdhani-Medium.ttf`
- `Rajdhani-SemiBold.ttf`
- `Rajdhani-Bold.ttf`

Cómo conseguirlos:

1. Ir a https://fonts.google.com/specimen/Rajdhani
2. "Download family" → descomprimir el zip.
3. Copiar los 3 TTF de arriba a esta carpeta (`public/fonts/`).

Licencia: SIL OFL 1.1 (libre para empaquetar en la app, también comercialmente).
El zip incluye el fichero `OFL.txt` — cópialo también aquí como atribución.
